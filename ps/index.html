2020 - NLP.Studio

Text analysis tools, including the world's most powerful, accurate, and comprehensive thesaurus. Why a thesaurus? Because I set out to build a simple domain suggestion tool. However, I soon realized that it was not so easy to generate meaningful phrases which are not awkward and not offensive. It's also difficult to parse strings of characters with no spaces. However, the main goal of this venture was to end up with some sort of intellectual property. So, I started compiling my own dictionary. I figured if nothing, I could sell the data to some AI company. In the end, it came out so good that I decided to start my own little AI company!

2008 - User Friendly CMS
Before Wordpress re-designed their UI in 2009, it was difficult to use, though not as difficult as Joomla/Droopal/Typo3. Back then it was an tedious work for independent website owners to manage their site using a full featured CMS. I designed and developed this "easy" CMS for my clients. Made with PHP/MySQL on the backend and jQuery UI on the front. It is built into the site itself. It uses custom html tags that lay dormant until the user signs in. Since then, most of my clients migrated to SquareSpace and Shoppify. But, the CMS and demo site are still running.
Demo: http://userfriendlycms.artspaces.net »
